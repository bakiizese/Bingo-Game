# 🎱 Bingo Game (React Native)

A simple and interactive **Bingo mobile game** built with **React Native**.
This project focuses on core game logic, state management, and clean mobile UI design.

## 🎮 Features

* 🧾 Dynamic Bingo card generation
* 🎲 Random number drawing system
* ✔️ Real-time number marking on the card
* 🏆 Automatic win detection (row, column, diagonal)
* 📱 Responsive and mobile-friendly UI
* 🔄 Smooth state updates using React Hooks
* 🎯 Simple and intuitive gameplay experience

---

## 🛠️ Tech Stack

* React Native
* JavaScript (ES6+)
* React Hooks (useState, useEffect)
* Expo (optional, if used)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/bingo-game.git
cd bingo-game
```

### 2. Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

### 3. Run the app

If using Expo:

```bash
npx expo start
```

If using React Native CLI:

```bash
npx react-native run-android
# or
npx react-native run-ios
```

## 🧠 How It Works

* A Bingo card is generated with random numbers
* The app randomly draws numbers one by one
* Matching numbers are automatically marked
* The game checks for winning patterns continuously
