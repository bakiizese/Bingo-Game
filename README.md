# 🎱 Bingo Game (React Native)

A simple and interactive **Bingo game mobile app** built using **React Native**.  
This project focuses on game logic, UI design, and state management in a mobile environment.

---

## 🎮 Features

- 🧾 Automatic Bingo card generation
- 🎲 Random number selection system
- ✔️ Real-time number marking on the card
- 🏆 Win condition detection
- 📱 Clean and responsive mobile UI
- 🔄 Smooth state updates during gameplay
- 🎯 Simple and intuitive user experience

---

## 🛠️ Tech Stack

- **React Native**
- **JavaScript**
- **React Hooks (useState, useEffect)**
- **Expo (if used)**
